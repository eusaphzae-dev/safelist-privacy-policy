1. Introduction

SafeList AliExpress is a browser extension designed to help users collect and export product information from e-commerce platforms such as eBay and AliExpress to automation tools like Make.com. We value your privacy and are committed to transparency and compliance with the Chrome Web Store User Data Policy.

2. Information We Collect

2.1 Current Version (No Personal Data Collected)

The current version of SafeList AliExpress does not collect, store, or process any personal data. This includes:

  • Email addresses
  • Names
  • Payment information
  • Login credentials
  • Browsing history
  • Cookies
  • IP address
  • Any personally identifiable information (PII)

The extension only processes non-personal, public product information that users choose to extract from e-commerce pages.

2.2 Future Versions (If Login or Account Features Are Added)

Future versions of SafeList AliExpress may introduce optional account features that require user login. If implemented, the extension may collect:

  • Email address (for authentication)
  • Profile information voluntarily provided
  • Subscription or payment status (for premium features)

Any such changes will be clearly communicated, and this Privacy Policy will be updated before those changes take effect.

3. Data Usage

Current Behavior

The extension only sends product information to a webhook URL (e.g., Make.com) that the user personally configures. No personal data is transmitted.

If Login/Accounts Are Introduced Later

If account features are added in future versions, user data such as email may be used for:

  • User authentication
  • Access to premium features
  • Saving user preferences
  • Communication about your account

We will never sell, trade, or share personal user information with third parties.

4. Data Storage

Current Version

The extension does not store any user data in:

  • chrome.storage
  • LocalStorage
  • Cookies
  • External servers

Future Storage (If Account System Is Added)

If future versions require saving user preferences or authentication tokens, this will be clearly stated in an updated version of this Privacy Policy.

5. Third-Party Services

Make.com / Webhooks

The extension may send non-personal product data to Make.com or similar automation platforms only when configured by the user. We do not store or access this data.

Analytics / Authentication (Future)

If analytics tools or authentication services (such as Firebase or Supabase) are added later, users will be notified within the extension, and this Privacy Policy will be updated accordingly.

6. Children’s Privacy

SafeList AliExpress is not intended for children under 13. We do not knowingly collect personal data from children.

7. Your Rights

If future versions of the extension collect personal data, users will have the right to:

  • Access their personal data
  • Modify or correct their data
  • Request deletion of their data
  • Opt out of optional data collection

Any such requests may be sent to: eusaphzae@gmail.com
